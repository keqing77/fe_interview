##### 什么是事件循环？

- 执⾏同步代码，这属于宏任务
- 执⾏栈为空，查询是否有微任务需要执⾏
- 执⾏所有微任务
- 必要的话渲染 UI
- 然后开始下⼀轮 Event loop，执⾏宏任务中的异步代码
